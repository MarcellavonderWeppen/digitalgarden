# Euria - eine ethisches KI

Euria wurde als datenschutzfreundliche Alternative zu ChatGPT entwickelt. 

Das System wird vollständig in der Schweiz betrieben und nutzt ein nachhaltiges Kühlkonzept, bei dem die Abwärme der Server direkt zum Beheizen von rund 6.000 Wohnungen in Genf genutzt wird.

Die Firma dahinter ist [[02 - Künstliche Intelligenz/2- Veröffentlicht/Infomaniak\|Infomaniak]]: eine europäischen Alternative zu den US-amerikanischen [[02 - Künstliche Intelligenz/2- Veröffentlicht/Hyperscaler\|Hyperscalern]]. 

Im Mai 2026 habe ich meinen bezahlten [[02 - Künstliche Intelligenz/1 - Work on next/ChatGPT\|ChatGPT]]-Account gekündigt und mich für ein Experiment entschieden: Euria sollte einen Monat lang meine neue Haupt-KI werden.
