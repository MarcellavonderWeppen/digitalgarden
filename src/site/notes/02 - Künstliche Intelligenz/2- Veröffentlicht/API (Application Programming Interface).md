# API (Application Programming Interface)

👉 Auf Deutsch: **Programmierschnittstelle**

Eine Schnittstelle ist der Ort der Kommunikation. 

Folglich ist eine API ist der Ort, an dem sich zwei Computerprogramme begegnen und Daten austauschen können. 

Typischerweise holen sich Apps Informationen (Daten) **von Servern, welche auf Datenbanken zugreifen**.

### Beispiel:

- Eine Wetter-App nutzt eine API, um Daten von einem Wetterdienst abzurufen.
- Eine schöne Benutzeroberfläche zeigt die aktuelle Temperatur, aber im Hintergrund spricht die App über die API mit einem Server.
