# CSS (Cascading Style Sheets)

CSS ist das schicke Geschwisterchen von [[02 - Künstliche Intelligenz/2- Veröffentlicht/HTML (Hypertext Markup Language)\|HTML]]. Es ist für das Design von Webseiten zuständig.

Ohne CSS würden das gesamte Internet noch immer wie in den 90ern aussehen:

![Website without HTML 2.png](/img/user/02%20-%20K%C3%BCnstliche%20Intelligenz/1%20-%20Work%20on%20next/Bilder/Website%20without%20HTML%202.png)

Gut, dass wir CSS erfunden haben!

## Ein einfaches Beispiel

So könnte eine CSS-Regel aussehen:

```CSS
h1 {
color: blue;
font-size: 20px;
}
```

Das bedeutet: "Mache alle Hauptüberschriften (`h1`) blau und gib ihnen eine Größe von 20 Pixeln."

## Warum zwei Sprachen?

An diesem Punkt stellt sich vielleicht die Frage: Wozu zwei Programmiersprachen - HTML _und_ CSS? Hätte man nicht auch eine Sprache erfinden können, die alles kann?

Das ist eine hervorragende Frage und die Antwort lautet: Die Trennung von Struktur und Aussehen ist ein genialer Schachzug.

Man stelle sich eine Webseite mit hunderten Unterseiten vor. Jede Unterseite hat eine große blaue Überschrift wie in unserem Beispiel oben. Nun entscheidet der Besitzer der Website: Die Überschriften sollen rot sein!

Wären HTML und CSS in einer Sprache verschmolzen, müssten wir nun auf hunderten Seitenvon Hand die Überschrift von blau auf rot umstellen.

Aber zum Glück gibt es CSS!

Eine einzige CSS-Datei kann mit beliebig vielen Unterseiten verknüpft sein.

Wir ändern einfach ein einziges einmal die CSS-Regel:

```CSS
h1 {
color: red;
font-size: 20px;
}
```

Fertig!
